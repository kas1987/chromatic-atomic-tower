# Evidence

Evidence is proof that work happened and was validated.

Use these folders:

| Folder | Use |
|---|---|
| `logs/` | Runtime or agent logs |
| `test-results/` | Test output |
| `diffs/` | Patch or diff evidence |
| `screenshots/` | Visual evidence |
| `reports/` | Markdown summaries, validation reports, closeouts |

Do not close BEADs or missions without evidence unless the closeout explicitly states why evidence is not applicable.
