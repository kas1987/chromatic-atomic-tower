# Mission Complexity Matrix

## M1 Basic

Simple, low-risk, well-defined work.

## M2 Intermediate

Multiple steps, moderate risk, limited dependencies.

## M3 Complex

Multiple systems, higher risk, broader validation required.

## M4 Atomic

Critical, high-risk, irreversible or far-reaching work. Requires maximum governance.

## CAT extension

CAT adds two required axes:

- Risk
- Reversibility

Autonomy is determined by complexity, risk, reversibility, confidence, and validation quality.
