# Evidence Capture Skill

## Intent

Make agent behavior deterministic, reviewable, and bounded.

## Inputs

- Mission Pack
- BEAD contract
- CAT manifest
- Current repo state
- Applicable gates

## Steps

1. Confirm mission and BEAD identifiers.
2. Confirm allowed and forbidden paths.
3. Identify assertion being addressed.
4. Execute only the approved procedure.
5. Capture direct evidence.
6. Report exceptions and next actions.

## Evidence

- execution log
- changed file list
- validation result
- exception log if applicable
- closeout summary
