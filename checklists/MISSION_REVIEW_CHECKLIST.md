# Mission Review Checklist

- [ ] Mission ID is valid
- [ ] Complexity level is correct
- [ ] Risk and reversibility are scored
- [ ] Scope in/out is clear
- [ ] Allowed paths are explicit
- [ ] Forbidden paths are explicit
- [ ] Validation is possible
- [ ] Rollback is defined if needed
- [ ] Human gate is correct
- [ ] BEAD list exists or is intentionally empty
