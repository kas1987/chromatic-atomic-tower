# CAT Audit Harness Playbook

## Operating sequence

```text
1. Capture intent
2. Build mission pack
3. Classify M1-M4 and C1-C4
4. Decompose into BEADs
5. Dispatch scoped agent
6. Capture evidence through magnets
7. Run substantive validation
8. Run control validation
9. Score confidence
10. Close, learn, and queue next work
```

## Audit-to-Harness rules

- Completeness before correctness.
- Controls do not replace substantive testing.
- Evidence quality scales with mission risk.
- Exceptions must be disclosed, not hidden.
- Learning is required before the next loop.
