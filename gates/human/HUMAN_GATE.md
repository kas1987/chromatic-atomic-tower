# Human Gate

Human approval is required for:

- M4 missions
- production deployment
- secret handling
- destructive file or data operations
- infrastructure changes
- public release
- irreversible changes
- cross-repo mutation
- costly external services

## Human gate request format

```md
## Human Gate Request

Mission:
BEAD:
Decision needed:
Recommended option:
Alternatives:
Risk:
Evidence:
Consequence of no action:
```
