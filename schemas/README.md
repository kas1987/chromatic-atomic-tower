# Schemas

CAT uses schemas to keep mission and BEAD contracts executable.

Run:

```bash
python scripts/cat_validate.py --all
```

Schema files:

- `mission.schema.json`
- `bead.schema.json`
- `mission_registry.schema.json`
- `agent.schema.json`
- `evidence.schema.json`
- `mission_log.schema.json`
- `tower_state.schema.json`
- `telemetry_envelope.schema.json` (LOGHOUSE)
- `finding.schema.json` (LOGHOUSE)
- `dependency_edge.schema.json` (LOGHOUSE)
- `deploy_event.schema.json` (LOGHOUSE)
